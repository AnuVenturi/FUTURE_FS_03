# SVS Boutique - Custom Stitching & Embroidery Website

A complete full-stack website for SVS Boutique, a custom stitching and embroidery business in Bhimavaram.

## 🎯 Features

- 🛍️ Customer Order Placement with multiple outfit selection
- 🔐 Admin Panel with secure login
- 📊 Order Management (Pending/Delivered)
- 🖼️ Sample Photos Gallery
- 📱 Fully Responsive Design
- 💾 MongoDB Atlas Database Integration

## 🛠️ Technologies Used

- **Frontend:** HTML5, CSS3, JavaScript
- **Backend:** Node.js, Express.js
- **Database:** MongoDB Atlas
- **Icons:** Font Awesome

## 🔧 Installation

### Prerequisites
- Node.js installed
- MongoDB Atlas account

### Backend Setup

```bash
cd backend
npm install
npm run dev
